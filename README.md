# 🃏 Санжут — Telegram Mini App

## Шаг 1: Установи Node.js

1. Зайди на https://nodejs.org
2. Скачай версию **LTS** (большая зелёная кнопка)
3. Установи как обычную программу (просто жми "Далее")

---

## Шаг 2: Загрузи проект на GitHub

1. Зайди на https://github.com и зарегистрируйся (если нет аккаунта)
2. Нажми **"New repository"** (зелёная кнопка)
3. Назови репозиторий: `sanzhut`
4. Нажми **"Create repository"**
5. Нажми **"uploading an existing file"**
6. Перетащи ВСЕ файлы из этой папки (включая папку `src`)
7. Нажми **"Commit changes"**

---

## Шаг 3: Задеплой на Vercel (бесплатно)

1. Зайди на https://vercel.com
2. Нажми **"Sign Up"** → войди через GitHub
3. Нажми **"Add New Project"**
4. Выбери репозиторий `sanzhut`
5. Настройки оставь по умолчанию, нажми **"Deploy"**
6. Через ~1 минуту получишь ссылку вида: `https://sanzhut-xxx.vercel.app`

**Скопируй эту ссылку — она понадобится на следующем шаге!**

---

## Шаг 4: Создай Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Отправь команду: `/newbot`
3. Придумай название бота (например: `Санжут Игра`)
4. Придумай username (например: `SanzhutGameBot`)
5. BotFather даст тебе **токен** — сохрани его

---

## Шаг 5: Привяжи Mini App к боту

В чате с @BotFather:

1. Отправь `/newapp`
2. Выбери своего бота
3. Введи название: `Санжут`
4. Отправь любую картинку (иконка игры)
5. Введи GIF или напиши "empty" (можно пропустить)
6. Введи свою ссылку Vercel: `https://sanzhut-xxx.vercel.app`
7. Введи короткое имя: `play`

---

## Готово! 🎉

Теперь в боте будет кнопка **Играть**. Поделись ссылкой с друзьями:
```
https://t.me/ИМЯ_ТВОЕГО_БОТА/play
```

Скинь эту ссылку в общий чат — каждый откроет игру прямо в Telegram!
